# VATools 使用指南

## 音频编辑功能使用说明

### 1. 上传音频文件
- 进入"音频编辑"标签页
- 拖拽或点击上传音频文件（支持 MP3, WAV, FLAC）
- 上传成功后会显示波形图

### 2. 波形选择与播放

#### 如何选择片段：
1. **在波形上拖拽鼠标**：按住鼠标左键，在波形区域拖拽
2. **选区会以半透明蓝色显示**
3. **选区信息**：
   - 顶部显示总时长和当前位置
   - 选区下方显示"已选择片段"提示

#### 如何播放选区：
1. 选择好片段后，点击 **"播放选区"** 按钮
2. 系统会只播放你选择的那段音频
3. 点击 **"停止"** 按钮可以随时停止播放

#### 使用提示：
- 💡 波形上方有使用提示
- 💡 选区可以拖动调整边界
- 💡 选区可以整体拖动移动位置
- 💡 双击选区可以删除它

### 3. 编辑操作

#### 提取片段：
1. 选择想要的片段
2. 选择"提取片段"操作
3. 选择输出格式（MP3/WAV/FLAC）
4. 点击"提取片段"按钮
5. 下载结果文件

#### 删除片段：
1. 选择不想要的片段
2. 选择"删除片段"操作
3. 选择输出格式
4. 点击"删除片段"按钮
5. 下载结果文件（删除了选中部分）

### 4. 常见问题

**Q: 为什么点击"播放选区"没反应？**
A: 请确保已经在波形上拖拽选择了片段，选区会显示为半透明蓝色。

**Q: 如何调整选区范围？**
A: 
- 方法1：拖拽选区的左右边缘来调整
- 方法2：重新拖拽选择新的区域

**Q: 如何取消选区？**
A: 双击选区即可删除，然后可以重新选择。

**Q: 选区太小看不到？**
A: 
- 使用波形右上角的缩放功能放大
- 调整浏览器窗口大小

### 5. 键盘快捷键

- **空格键**：播放/暂停（在波形区域获得焦点时）
- **Esc键**：停止播放

## 音频提取功能

### 使用步骤：
1. 进入"音频提取"标签页
2. 上传视频文件（支持 MP4, AVI, MOV, MKV）
3. 选择输出格式（MP3/WAV/FLAC）
4. 选择比特率（建议 192k）
5. 点击"开始提取"
6. 等待处理完成后下载或进入编辑

## 音源分离功能

### ⚠️ 功能限制
音源分离功能需要 Python 3.9-3.11 环境。如果您的 Python 版本不匹配，此功能将不可用。

### 使用步骤：
1. 进入"音源分离"标签页
2. 上传音频文件
3. 勾选要分离的音轨：
   - 人声
   - 鼓点
   - 贝斯
   - 其他伴奏
4. 点击"开始分离"
5. 等待处理完成后下载分离后的音轨

## 设置功能

在"设置"标签页可以配置：
- 工作目录（上传、输出、日志）
- 日志级别和保留策略
- 音频默认参数（格式、比特率、采样率）
- 音源分离加速方式（MPS/MLX/CPU）
